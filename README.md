# BEFITWITHJEMAL GYM
BefitwithJemal Gym-milestone-project-1 is a fully responsive multi-page fitness website developed for the User Centric Frontend Development Milestone Project, as part of the Level 5 Diploma in Web Application Development.
The project demonstrates semantic HTML5 structure, professional CSS styling, responsive design principles, accessibility awareness.


##  Project Overview

The website represents a modern fitness centre offering:

- Professional training services
- Weekly class schedule
- Modern gym facilities
- Community-focused environment
- Contact form for enquiries
- Google Maps integration

## User Experience (UX)
### Project Goals
- Provide clear and intuitive navigation
- Present gym services in a structured format
- Maintain consistent branding and colour scheme
- Ensure accessibility and readability
- Deliver responsive performance across devices
- Target Audience
- Individuals seeking fitness training
- Beginner and intermediate gym members
- People looking for personal coaching
- Community-focused fitness enthusiasts

## Features

- Fully responsive layout
- Wide screen layout (max-width 1200px)
- Colourful modern UI design
- Hero sections with gradients
- Weekly classes schedule table
- Contact form
- Embedded Google Map
- Image gallery with captions
- Clean folder structure
- Semantic HTML5 elements


## Technologies Used

- HTML5
- CSS3
- VSC
- GitHub
- Google Maps Embed
- Chrome Developer Tools: Device Views to test
